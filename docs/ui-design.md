# UIデザイン仕様書: GitHub PR Dashboard

## 1. タイポグラフィ

### 1.1 フォントファミリー

| 用途               | フォント                                                                                   |
| ------------------ | ------------------------------------------------------------------------------------------ |
| UI 全般            | -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif |
| コード・ブランチ名 | "SF Mono", "Cascadia Code", "Fira Code", Consolas, "DejaVu Sans Mono", monospace           |

### 1.2 サイズスケール

| トークン  | サイズ | 用途                                 |
| --------- | ------ | ------------------------------------ |
| text-xs   | 11px   | 補足テキスト、タイムスタンプ         |
| text-sm   | 13px   | 副テキスト、ブランチ名、バッジラベル |
| text-base | 14px   | 本文、リスト項目、入力フィールド     |
| text-md   | 16px   | サイドバー見出し、PR タイトル        |
| text-lg   | 20px   | セクション見出し(リポジトリ名)       |
| text-xl   | 24px   | ページ見出し(ヘッダーロゴ横)         |
| text-2xl  | 32px   | セットアップ画面タイトル             |

### 1.3 ウェイト

| トークン      | 値  | 用途                                   |
| ------------- | --- | -------------------------------------- |
| font-normal   | 400 | 本文、補足テキスト                     |
| font-medium   | 500 | ブランチ名、バッジ、ナビゲーション項目 |
| font-semibold | 600 | PR タイトル、セクション見出し          |
| font-bold     | 700 | ページタイトル、セットアップ画面見出し |

### 1.4 行高

| トークン        | 値   | 用途   |
| --------------- | ---- | ------ |
| leading-tight   | 1.25 | 見出し |
| leading-normal  | 1.5  | 本文   |
| leading-relaxed | 1.75 | 説明文 |

## 2. スペーシング

### 2.1 スケール (4px ベース)

| トークン | 値   | 主な用途                                       |
| -------- | ---- | ---------------------------------------------- |
| space-1  | 4px  | アイコンとテキストの間                         |
| space-2  | 8px  | バッジ内パディング、要素間の最小間隔           |
| space-3  | 12px | カード内パディング(小)、リスト項目間           |
| space-4  | 16px | カード内パディング(標準)、セクション内の要素間 |
| space-5  | 20px | カード内パディング(大)                         |

| space-8  | 32px | メインエリアのパディング                       |
| space-10 | 40px | セクション間(大)                               |
| space-12 | 48px | ページ上下のパディング                         |

### 2.2 レイアウトスペーシング

| 要素                 | パディング                       | マージン          |
| -------------------- | -------------------------------- | ----------------- |

### 4.2 PR カード

1つの PR を表す情報単位。

```
┌────────────────────────────────────────────────────────────┐
│ #123  Add new feature                              2日前   │  ← 行1: PR番号 + タイトル(リンク) + 作成日時
│                                                            │
│ feature/new-thing                            更新: 1時間前  │  ← 行2: ブランチ名(monospace) + 更新日時
│                                                            │
│ [avatar] user1    →  [avatar] user2     [Approved]         │  ← 行3: 作成者 → アサイン先 + レビューバッジ
└────────────────────────────────────────────────────────────┘
```

| プロパティ       | 値                                |
| ---------------- | --------------------------------- |
| 背景             | bg-card (#262626)                 |
| ボーダー         | 1px solid border-subtle (#333333) |
| 角丸             | 8px                               |
| パディング       | space-4 (16px)                    |
| カード間マージン | space-3 (12px)                    |
| ホバー           | ボーダー #444444, 背景 #2a2a2a    |

#### 行1: ヘッダー行

| 要素           | スタイル                                                                |
| -------------- | ----------------------------------------------------------------------- |
| PR 番号 (#123) | text-sm, font-medium, text-secondary                                    |
| タイトル       | text-md, font-semibold, text-link, カーソル pointer, ホバー時 underline |
| 作成日時       | text-xs, text-tertiary, 右寄せ                                          |

#### 行2: ブランチ行

| 要素       | スタイル                                                                                    |
| ---------- | ------------------------------------------------------------------------------------------- |
| ブランチ名 | text-sm, font-mono, font-medium, text-secondary, 背景 #333333, パディング 2px 6px, 角丸 4px |
| 更新日時   | text-xs, text-tertiary, 右寄せ                                                              |

#### 行3: メタ行

| 要素           | スタイル                                 |
| -------------- | ---------------------------------------- |
| 作成者         | アバター(20px) + text-sm, text-secondary |
| 矢印 (→)       | text-xs, text-tertiary                   |
| アサイン先     | アバター(20px) + text-sm, text-primary   |
| 未アサイン     | text-sm, text-tertiary, イタリック       |
| レビューバッジ | 右寄せ (4.4 参照)                        |
